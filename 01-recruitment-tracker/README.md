# 📋 Project 01 — Recruitment Tracker

## Overview

A comprehensive **Recruitment Tracker** built in Excel to manage and monitor the end-to-end hiring process. Designed for high-volume recruitment scenarios where visibility into the candidate pipeline is critical.

---

## 🎯 Problem Statement

Recruitment teams often lose track of candidates across multiple stages, leading to delayed hiring decisions and poor candidate experience. This tracker was built to solve that by creating a single source of truth for the entire hiring funnel.

---

## 🔑 Key Features

- **Candidate Pipeline Management** — Tracks candidates from application through to final offer
- **Automated Status Tracking** — Dropdown menus to categorise progress: Screened → Interviewed → Offered → Rejected
- **KPI Ready** — Built to calculate:
  - ⏱️ Time-to-Hire
  - 🔍 Source Effectiveness
  - 📉 Drop-off Rate by Stage
